== Installation ==

1. From the 'Plugins' menu in WordPress click 'Add New' then upload and then using the form upload 'current-star-sign.zip' or you can search for Current Star Sign in the search box and install from there.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets and drag the Current Star Sign widget to where you want it to display in your sidebar.

To install the plugin manually, follow these steps:

1. Upload the entire 'current-star-sign.zip' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Widgets and drag the Current Star Sign widget to where you want it to display in your sidebar.
